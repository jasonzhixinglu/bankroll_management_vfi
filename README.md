# Poker Optimal Bankroll Management

A self-contained Python implementation of the Optimal Bankroll Management
(OBM) framework. The model frames game selection as a dynamic programming
problem and derives the bankroll thresholds at which a player should move
up to higher stakes — without relying on risk-of-ruin heuristics or the
Kelly criterion.

## Models

**Finite-horizon DP** (`brm.finite_dp`)
Backward induction over T periods. Each period the player chooses from N
discrete games; transitions follow a Bernoulli stack-off rule. The
terminal value is risk-neutral (V_T = x), and the implied risk aversion
generated by the *optionality* of higher-stakes games is recovered via
`calc_eta`.

**Infinite-horizon OBM** (`brm.infinite_obm`)
Value iteration converging on CEW (Certainty-Equivalent Winrate) distance.
Next-period bankrolls are sampled with 7-point normal quadrature; the
value function is interpolated log-linearly. Benchmarked against the
one-shot Kelly criterion (`brm.kelly`).

## Quickstart

```bash
# 1. Create and activate the conda environment
conda env create -f environment.yml
conda activate brm

# 2. Install the package in editable mode
pip install -e .
```

```python
from brm import gen_xgrid, solve_obm, extract_threshold

game_choices = {
    0: (0,   5,   0),    # outside option (work): no buyin, $5/hr, no variance
    1: (100, 10, 100),   # 1/2 game
    2: (200, 15, 200),   # 2/5 game
    3: (500, 20, 500),   # 5/10 game
}

xgrid = gen_xgrid(xmax=500_000, xmin=100, gridsize=100)
CEW_array, CEW_choices, vf = solve_obm(game_choices, xgrid)

for n in range(1, len(game_choices)):
    thresh = extract_threshold(n, CEW_choices, xgrid, game_choices)
    print(f"Move up to game {n} at bankroll: ${thresh:,.0f}")
```

## Project layout

```

  __init__.py      public API
  grid.py          gen_xgrid, sample_loglin_vf, build_sampling_df
  finite_dp.py     OptimalBRM (backward induction), calc_eta
  kelly.py         one-shot Kelly CEW benchmark
  cew.py           compute_cew — Newton solve for CEW per (x, game) row
  infinite_obm.py  solve_obm — value iteration to convergence
  thresholds.py    extract_threshold — zero-crossing of CEW difference
```
